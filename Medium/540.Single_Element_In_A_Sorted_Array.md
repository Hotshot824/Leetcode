### [540. Single Element in a Sorted Array]

A single for loop solution, time complexity is O(n/2).
```go
func singleNonDuplicate(nums []int) int {
	var i int = 0
	for ; i < len(nums)-1; i += 2 {
		if nums[i] != nums[i+1] {
			return nums[i]
		}
	}
	return nums[i]
}
```
