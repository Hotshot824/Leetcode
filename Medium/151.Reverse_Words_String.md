### [151. Reverse Words in a String]

>Tip. This is solution using split. Don't use it if you challenge yourself.

```go
func reverseWords(s string) string {
	var result string
	ss := strings.Fields(s)
	for i := len(ss) - 1; i >= 0; i-- {
		tmp := strings.Replace(ss[i], " ", "", -1)
		result += tmp
		if i-1 >= 0 {
			result += " "
		}
	}
	return result
}
```
