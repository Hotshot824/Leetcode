### [268. Missing Number]

Reference [41. First Missing Positive] to solve. This time complexity is O(n).

```go
func missingNumber(nums []int) int {
	var maxint int = math.MinInt
	hash := map[int]bool{}
	for _, num := range nums {
		if num > maxint {
			maxint = num
		}
		hash[num] = true
	}
	var i = 0
	for i <= maxint {
		ok, _ := hash[i]
		if !ok {
			return i
		}
		i++
	}
	return i
}
```
