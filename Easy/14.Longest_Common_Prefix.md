### [14. Longest Common Prefix]

比較笨的方法，每個 String 逐一比對，直到有一個 String 結束或是遇到不同的字元。

```go
func longestCommonPrefix(strs []string) string {
	var result string = ""
	var c byte
	for i, j := 0, 0; true; i++ {
		if i >= len(strs) {
			i = 0
			result += string(c)
			j++
		}
		if j >= len(strs[i]) {
			break
		}
		if i == 0 {
			c = strs[i][j]
		}
		if strs[i][j] != c {
			break
		}
	}
	return result
}
```
