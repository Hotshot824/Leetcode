### [344. Reverse String]

simple problem, change the first and last index, and towards the midindex to exchange all element.

```go
func reverseString(s []byte) {
	for i, j := len(s)-1, 0; j < i; {
		s[i], s[j] = s[j], s[i]
		i--
		j++
	}
	return 
}
```
