### [977. Squares of a Sorted Array]

Do the power of all array elements first, then using go sort package sorted.
The time complexity is O(n).

```go
func sortedSquares(nums []int) []int {
	for i := 0; i < len(nums); i++ {
		nums[i] = Pow(nums[i])
	}
	sort.Ints(nums)
	return nums
}

func Pow(x int) int {
	return x * x
}
```
