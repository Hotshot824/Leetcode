### [69. Sqrt(x)]

Sqrt 也可以使用 Binary search 來做, 概念與猜數字類似, 先從中間值開始猜，
然後分左右區段, 當 mid**2 大於或小於 x 時候向左或向右移動，直到找到相符的數字.

```go
func Pow(x int) int {
	return x * x
}

func mySqrt(x int) int {
	left, right := 0, x
	for left <= right {
		mid := left + (right-left)/2
		product := Pow(mid)
		if product > x {
			right = mid
		} else {
			if (mid == x) || Pow(mid+1) > x {
				return mid
			}
			left = mid + 1
		}
	}
	return -1
}

```
